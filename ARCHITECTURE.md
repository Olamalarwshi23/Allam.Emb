# بنية Allam EMB Studio

## نظرة عامة

Allam EMB Studio هو محرك متقدم لتصميم وتحرير ملفات التطريز، يجمع بين واجهة ويب سلسة وتطبيق أندرويد احترافي.

## المكونات الرئيسية

### 1. محرك التطريز (Embroidery Engine)
- **معالجة الغرز**: توليد غرز الساتان والتاتامي والقفز
- **حساب المتجهات**: دقة عالية للإحداثيات
- **إدارة الألوان**: لوحة ألوان متكاملة

### 2. معالج الملفات
- **استيراد**: قراءة ملفات EMB و DST
- **تصدير**: توليد ملفات DST قابلة للاستخدام على الماكينات
- **التخزين**: حفظ التصاميم محلياً

### 3. واجهة المستخدم
- **لوحة التحكم**: أدوات الرسم والتحرير
- **لوحة الألوان**: اختيار الخيوط
- **الإحصائيات**: عداد الغرز والمعلومات

## تدفق البيانات

```
رسم المستخدم → معالجة النقاط → توليد الغرز → الرسم على Canvas → عرض الإحصائيات
```

## ملفات الملفات

### DST (Tajima)
- **الترويسة**: 512 بايت
- **بيانات الغرز**: 3 بايتات لكل حركة
- **النهاية**: 0x00 0x00 0xF3

### EMB (Elna)
- **الترويسة**: 1024 بايت
- **بيانات الغرز**: 3 بايتات لكل حركة
- **معالجة خاصة**: توافق مع ماكينات Wilcom

## خوارزميات الغرز

### غرز الساتان (Satin)
- حساب الزاوية بين النقاط
- توليد خطوط متوازية على جانبي المسار
- كثافة قابلة للتعديل

### حشو التاتامي (Tatami)
- حساب الصندوق المحيط
- ملء أفقي مع نمط التبديل
- منع التشقق والفجوات

### غرز القفز (Jump)
- خطوط منقطة وهمية
- لا تؤثر على النتيجة النهائية
- للفصل بين الأشكال
