# 🪡 Allam EMB Studio - تطبيق التطريز الاحترافي

**أناملنا تصنع شعاعا من خيوط الشمس** ✨

تطبيق أندرويد متقدم وحديث لتصميم وتحرير ملفات التطريز بسهولة واحترافية.

## الميزات الرئيسية

### 🎨 محرر التطريز المتقدم
- **غرز متعددة**: ساتان (Satin)، تاتامي (Tatami)، قفز (Jump)
- **توليد غرز ذكي**: محرك متجهي متقدم لحساب الإحداثيات
- **لوحة ألوان غنية**: تشكيلة كاملة من خيوط الألوان

### 📁 إدارة الملفات
- استيراد ملفات EMB و DST من الماكينات
- تصدير مباشر للماكينات (توافق Dahao و Wilcom)
- حفظ التصاميم المخصصة

### 🔧 أدوات احترافية
- **تحكم ديناميكي**: كثافة الحشو وطول الغرزة قابلة للتعديل
- **معاينة مباشرة**: رؤية التغييرات حياً
- **شبكة مرجعية**: للدقة والمحاذاة

### 📊 إحصائيات التصميم
- عداد غرز فوري
- حساب استهلاك الخيط
- تقديرات وقت الخياطة

## التقنيات المستخدمة

### 🔧 للتطبيق
- **Kotlin** - لغة Android الحديثة
- **Jetpack Compose** - واجهة مستخدم ديناميكية
- **Material Design 3** - تصميم حديث وأنيق

### 🖥️ للويب
- **HTML5 Canvas** - رسومات عالية الأداء
- **JavaScript** - معالجة البيانات والغرز
